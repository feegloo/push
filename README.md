#### Terminal

`push foo`&nbsp;&nbsp;&nbsp;`// creates new repository on github, then pushes all changes from current directory`

#### Installation

`curl -s https://raw.githubusercontent.com/feegloo/push/master/install | bash && . ~/.profile`
