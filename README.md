### Usage

`push`&nbsp;&nbsp;&nbsp;`// creates new repository (named as current directory) on github, makes commit and pushes changes`

### Installation

`curl -s https://raw.githubusercontent.com/feegloo/push/master/install | bash && . ~/.profile`
